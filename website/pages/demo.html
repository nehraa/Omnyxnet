Quick Start

Get up and running in 3 simple steps.

1

Install

Install the CLI globally with npm.

$ npm install -g pangea-net
2

Start

Launch your node and join the network.

$ pangea start
3

Upload

Store files on the decentralized network.

$ pangea upload ./file.zip
